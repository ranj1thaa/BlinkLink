<h1>🔗 BlinkLink – Full Stack URL Shortener & Analytics Platform</h1>

<p>
Production-grade URL shortening platform with authentication, analytics tracking, caching, and DevOps automation.
</p>

</div>

---

## 🚀 Overview

<p>
BlinkLink is a full-stack URL shortening and analytics platform built with modern web technologies.
</p>

<ul>
<li>Secure authentication</li>
<li>URL shortening with custom codes</li>
<li>Real-time analytics tracking</li>
<li>Production-ready DevOps deployment</li>
</ul>

---

## ✨ Features

<h3>🔐 Authentication</h3>

<ul>
<li>JWT Authentication</li>
<li>HTTP-only cookie storage</li>
<li>Role-based access control</li>
</ul>

<h3>🔗 URL Shortening</h3>

<ul>
<li>Custom short codes</li>
<li>URL expiration support</li>
<li>QR code generation</li>
<li>Activate / deactivate URLs</li>
</ul>

<h3>📊 Analytics</h3>

<ul>
<li>Click tracking</li>
<li>Country analytics</li>
<li>Browser analytics</li>
<li>Device analytics</li>
<li>Timeline analytics</li>
</ul>

<h3>⚡ Performance Optimization</h3>

<ul>
<li>Redis caching for fast redirection</li>
<li>MongoDB indexing</li>
<li>TTL expiration for URLs</li>
</ul>

<h3>🛡 Security</h3>

<ul>
<li>Helmet security headers</li>
<li>Rate limiting</li>
<li>CORS protection</li>
<li>Password hashing</li>
</ul>

<h3>🐳 DevOps & Deployment</h3>

<ul>
<li>Docker containerization</li>
<li>Nginx reverse proxy</li>
<li>GitHub Actions CI/CD</li>
<li>Render deployment</li>
</ul>

---

## 🏗 Architecture

<pre>
Frontend (React + Vite)
↓
Nginx Reverse Proxy
↓
Backend API (Node + Express)
↓
MongoDB Atlas
↓
Redis Cache
</pre>

---

## 🛠 Tech Stack

<h3>Frontend</h3>

<p>
React | Vite | Bootstrap | Chart.js | Axios
</p>

<h3>Backend</h3>

<p>
Node.js | Express.js | MongoDB Atlas | Redis | JWT
</p>

<h3>DevOps</h3>

<p>
Docker | Docker Compose | Nginx | GitHub Actions | Render
</p>

---

## 📦 Installation

<h3>Backend</h3>

<pre>
cd backend
npm install
npm run dev
</pre>

<h3>Frontend</h3>

<pre>
cd frontend
npm install
npm run dev
</pre>
